### What I learned

I learned how to implement an responsive solution for scaling the size of the card and fonts to different screen sizes. By looking at the figma properties of the design it was easier to get the margins and sizes correctly. A simple prompt in cursor was enough to understand and implement the hover functionality to change the title color.

### Continued development

Responsive behavior makes the difference in user experience. I just focused on what was needed for the project but this seems to be a very important topic to make quality UIs
